●readme.txt
このフォルダーに収められているファイルは、日経BPムック「個人投資家のためのFinTechプログラミング」（著者：てばさき氏、森谷博之氏ほか）に掲載したサンプルプログラムです。

ムックの範囲を超える問い合わせやご質問にはお答えできません。

プログラムはプログラミング言語「Python」で記述されています。

誌面とはソースコードの内容やファイル名が異なる場合があります。一部のプログラムは実行しやすいように修正してあります。

「pXX-pXX.txt」といったファイルは、該当ページ間の本文に挿入したソースコードを抜き出してテキスト化したものです。「pXX」はXXページを意味します。

プログラムは単体で動作するものと、単体では動作せず、他のプログラムと組み合わせて動かすものがあります。

外部のWeb API（Webサービス）を利用するプログラムは、Web APIの提供者の都合あるいはネットワークの状況によって、正しく動作しない可能性があります。

Pythonのライブラリを適切にインストールしていないと、プログラムが動作しない可能性があります。

Web APIを使うプログラムや、乱数を使うシミュレーションのプログラムは、実行結果が誌面とは異なります。

仮想通貨取引所から取得する、ユーザーごとに異なるAPIキーとAPIシークレットを設定しないと動かないプログラムがあります。

以上、あからじめご了承ください。
